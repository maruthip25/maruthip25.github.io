How to fix the missing PNG's bug in Yoeman's Webapp Generator

If you are among the few who do their coding on windows (like me), and you try to use the amazing Yoeman as a part of you frontend devp. workflow (like me), then there is a chance you might have come across a small bug (with huge consequences) in it's WebApp geneator.

The Bug

As of version 0.4.9 (of the Webapp Generator), on running grunt build (on windows), your resulting dist folder ends up with exactly 0 PNG's exported from your src folder :P.

I don't know if its the same for GIF's.

The Fix

In your Gruntfile.js, look for the configuration of the imagemin task

Find something similar to :

imagemin: {
        dist: {
            files: [{
                expand: true,
                cwd: 'src/images',
                src: '{,*/}*.{gif,jpeg,png,jpg}',
                dest: 'dist/images'
            }]
        }
    }

Now, apparently this task fails to work for PNG files, so we remove 'png' from the list under src:.

But, we still have to copy the png files to dist right? The copy task will do this for us.

Find something similar to :

copy: {
        dist: {
            files: [{
                expand: true,
                dot: true,
                cwd: 'src',
                dest: 'dist',
                src: [
                    '*.{ico,png,txt}',
                    '.htaccess',
                    'images/{,*/}*.webp',
                    '{,*/}*.html',
                    'styles/fonts/{,*/}*.*'
                ]
            }]
        }

Now, add an entry for PNG's in the array under src:

'images/{,*/}*.png' //coz imagemin doesnt do pngs

You'll end up with something similar to

copy: {
        dist: {
            files: [{
                expand: true,
                dot: true,
                cwd: 'src',
                dest: 'dist',
                src: [
                    '*.{ico,png,txt}',
                    '.htaccess',
                    'images/{,*/}*.webp',
                    'images/{,*/}*.png', //coz imagemin doesnt do pngs
                    '{,*/}*.html',
                    'styles/fonts/{,*/}*.*'
                ]
            }]
        }

Now save your Gruntfile.js and try grunt build.

Done. That's pretty much it.
