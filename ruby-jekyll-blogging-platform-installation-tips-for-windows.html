Jekyll Blogging Platform Installation Tips for Windows

About a week ago, I decided to try out the ruby-based blogging platform (also actually a static site generator) named "Jekyll" .

I realised there are a few issues with it if you try to use it on Windows.

Lets just skip the "address-each-issue" part. I'll simply point out 4 important steps to get it Up-and-Running smoothly :P

  1. Install Ruby 1.9.3 instead of Ruby 2.1.0 If you are like me. You might always have the temptation of downloading the "latest" version of everything. Well, in this case, you might have to calm down a bit.

    Although 1.9.3 is like the older version, it is supported till February 23, 2015 and 2.1.0 seems to have a couple of issues, especially with windows. Not only jekyll, even the mighty famous web framework rails might give you trouble if you try to run it on Ruby 2.1.0 (on windows, for sure).

  2. Install Jekyll 1.4.2 instead of Jekyll 1.4.3 The reasons are similar. Just stick with 1.4.2 till things get cleared.

  3. Pygments - the syntax highlighter This actually works well with jekyll. But, you'll need Python to run it (obviously to even install it). So, incase you decide not to have it included, there is one extra thing you have to do. Go to the _config.yml file in your jekyll root and set the pygments variable to false.

  4. Install wdm ( Windows Directory Monitor ) I don't know if everyone will have this problem. But, for some strange reason, in my case, the gem installer didn't automatically install a dependency name wdm. Just to make sure, run

      gem list

and if you don't see something like

wdm (0.1.0)

(almost at the end, because the list is usually sorted alphabetically) , then you have to install it. Simply run

  gem install wdm

Done. That's pretty much it.
